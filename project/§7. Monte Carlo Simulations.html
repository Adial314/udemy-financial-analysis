§7. Monte Carlo Simulations¶

As a means of estimating future events, Monte Carlo simulations allow for researchers to study how different probabilities interact to form trees of outcomes. In other words, they allow for data scientists to study several scenarios.

Monte Carlo Simulation Theory¶

Monte Carlo simulations (MCS) attempt to model a system using a range of inputs from random variables (RVs). If a RV has a given standard deviation, the MCS will find best and worst case scenarios from within and without the standard deviation of each of its RVs. As the number of RVs increase, so too do the permutations which the MCS must consider.

The application of MCS results in a larger data set to study as new outcomes have been estimated using existing historical data. It is important to remember that MCS does not predict the future but simply provides an outcome range which encapsulates the most likely future outcomes.

Using MCS¶

Consider predicting the gross profit of a company using historical trends. We will synthesize data using normal distributions to represent the revenue streams of the company.

import numpy as np

import matplotlib.pyplot as plt
%matplotlib inline

# Ignore warnings
import warnings

# ---------------- SYNTHESIZING DATA ---------------- #

# Set seed
np.random.seed(0)

# Define Gaussian stats
revenue_mu  = 170   # Statistical mean
revenue_sigma = 20  # Statistical standard deviation

# Specify sample number
iterations = 1000

# Create distribution
revenue = np.random.normal(revenue_mu, revenue_sigma, iterations)


# ---------------- VISUALIZING DATA ---------------- #

# Plot distribution
plt.figure(figsize = (15, 6))
plt.plot(revenue)
plt.show()
